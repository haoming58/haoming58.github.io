## 1.1 移除元素

给你一个数组 nums 和一个值 val，你需要 原地 移除所有数值等于 val 的元素，并返回移除后数组的新长度。

不要使用额外的数组空间，你必须仅使用 O(1) 额外空间并原地修改输入数组。

元素的顺序可以改变。你不需要考虑数组中超出新长度后面的元素。

示例 1: 给定 nums = [3,2,2,3], val = 3, 函数应该返回新的长度 2, 并且 nums 中的前两个元素均为 2。 你不需要考虑数组中超出新长度后面的元素。

示例 2: 给定 nums = [0,1,2,2,3,0,4,2], val = 2, 函数应该返回新的长度 5, 并且 nums 中的前五个元素为 0, 1, 3, 0, 4。

你不需要考虑数组中超出新长度后面的元素。

### 1.1.1 思路

这里有两种双指针思路。

**思路一：快慢指针（同向）**

由于这里的标题直接是双指针，就直接开始使用这个指针进行代码实践。

- **快指针**：寻找新数组的元素，新数组就是不含有目标元素的数组，一个用于遍历
- **慢指针**：指向更新新数组下标的位置，一个用于新数组的下标

**思路二：相向双指针**

首尾呼应，用右边有用的数字，去填补左边需要删除的空位。

1. **左右包抄（初始化）**
   - 定义两个指针，`left` 站在数组的最前面（索引 0），`right` 站在数组的最后面（索引 n-1）
   - 两人准备相向而行，从两头向中间巡视整个数组

2. **左指针找"坑"（第一个 while 循环）**
   - 左指针 `left` 一路向右走，只要它脚下的数字不是目标值 `val`，说明这个数字是安全的，不用管它
   - 一旦 `left` 踩到了 `val`，立刻停下
   - 此时左边出现了一个需要被填补的**"坑"**（废弃位置）

3. **右指针找"替补"（第二个 while 循环）**
   - 右指针 `right` 一路向左退，只要它脚下的数字是目标值 `val`，说明这本来就是要扔掉的，不用管它
   - 一旦 `right` 踩到了一个不是 `val` 的正常数字，立刻停下
   - 此时右边找到了一个可以用来填坑的**"替补"**（有效数据）

4. **填坑换位（`if left < right` 判断与覆盖）**
   - **填坑**：执行 `nums[left] = nums[right]`，直接用右边的"替补"把左边的"坑"覆盖掉
   - **指针推进**：填完坑后，双方都完成了一次任务，各自走向下一个目标
     - `left += 1`：左指针的坑填满了，往右走去检查下一个未知元素
     - `right -= 1`：右指针的替补用过了，往左走去寻找新的"替补"

### 1.1.2 代码

**思路一：快慢指针（写法 A）**

```python
class Solution():
    def remove(self, nums: list[int], val: int) -> int:
        fast = 0  # 按照你的习惯命名，但它实际是负责写入的指针

        for i in range(len(nums)):
            if nums[i] != val:
                nums[fast] = nums[i]
                fast += 1  # 成功写入一个元素后，指针往前推一格

        return fast  # fast 最后的值，恰好就是有效元素的个数
```

**思路一：快慢指针（写法 B）**

```python
class Solution:
    def removeElement(self, nums: List[int], val: int) -> int:
        # 快慢指针
        fast = 0  # 快指针
        slow = 0  # 慢指针
        size = len(nums)
        while fast < size:  # 不加等于是因为，a = size 时，nums[a] 会越界
            # slow 用来收集不等于 val 的值，如果 fast 对应值不等于 val，则把它与 slow 替换
            if nums[fast] != val:
                nums[slow] = nums[fast]
                slow += 1
            fast += 1
        return slow
```

**思路二：相向双指针**

```python
# 相向双指针法
# 时间复杂度 O(n)
# 空间复杂度 O(1)
class Solution:
    def removeElement(self, nums: List[int], val: int) -> int:
        n = len(nums)
        left, right = 0, n - 1
        while left <= right:
            while left <= right and nums[left] != val:
                left += 1
            while left <= right and nums[right] == val:
                right -= 1
            if left < right:
                nums[left] = nums[right]
                left += 1
                right -= 1
        return left
```

## 1.3 替换数字

给定一个字符串 s，它包含小写字母和数字字符，请编写一个函数，将字符串中的字母字符保持不变，而将每个数字字符替换为number。

例如，对于输入字符串 "a1b2c3"，函数应该将其转换为 "anumberbnumbercnumber"。

对于输入字符串 "a5b"，函数应该将其转换为 "anumberb"

输入：一个字符串 s,s 仅包含小写字母和数字字符。

输出：打印一个新的字符串，其中每个数字字符都被替换为了number

样例输入：a1b2c3

样例输出：anumberbnumbercnumber

数据范围：1 <= s.length < 10000

### 1.3.1 思路

按 C 或 C++ 来看，首先要考虑使用多少空间——**核心思想是从后往前挪移**。如果从前往后写会覆盖后面的数据。

- 两个指针，一个指向**原数据末尾**，一个指向**扩容后数据末尾**
- 先扫描原串统计数字个数，按 1 个数字 → 6 个字符（"number"）扩容
- 从后往前遍历，遇到字母直接拷贝，遇到数字倒序填入 "number"

> 替换的时候，本身数字的位置也占了一位，所以切片对应的范围是 `s[j-5:j+1]`（左闭右开 6 个位置）。

### 1.3.2 代码

```python
class Solution:
    def replaceNumber(self, s: list) -> list:
        # 1. 统计数字出现的次数
        count = 0
        for char in s:
            if '0' <= char <= '9':
                count += 1

        # 记录原始长度
        old_len = len(s)
        # 假设将 1 个数字替换为 "number" (6个字符)，则每个数字需要额外 5 个空间
        extra_space = count * 5
        s.extend([''] * extra_space)

        # 2. 定义双指针
        # i 指向原始列表末尾，j 指向扩容后的列表末尾
        i = old_len - 1
        j = len(s) - 1

        # 3. 从后往前替换
        while i >= 0:
            if '0' <= s[i] <= '9':
                # 如果是数字，倒序填充 "number"
                for char in reversed("number"):
                    s[j] = char
                    j -= 1
            else:
                # 如果不是数字，直接复制
                s[j] = s[i]
                j -= 1
            i -= 1

        return s
```

## 1.4 反转字符串里面的单词

输入: "the sky is blue"
输出: "blue is sky the"

示例 2：
输入: "  hello world!  "
输出: "world! hello"
解释: 输入字符串可以在前面或者后面包含多余的空格，但是反转后的字符不能包括。

示例 3：
输入: "a good   example"
输出: "example good a"
解释: 如果两个单词间有多余的空格，将反转后单词间的空格减少到只含一个。

### 1.4.1 思路

由于输出后的字符串不能有多余的空格，因此在反转之前，先把空格删除——并且需要删除两个单词间多余的空格。

具体操作：先找到非空的单词然后填补上，后面的单词和单词之间填补一个空格。

### 1.4.2 代码

**思路一：双指针删除多余空格**

```python
def remove_extra_spaces(s: list) -> list:
    slow, fast = 0, 0
    n = len(s)

    while fast < n:
        # 1. 寻找单词：跳过所有空格
        if s[fast] != ' ':
            # 2. 如果不是第一个单词，在单词前补一个空格
            if slow != 0:
                s[slow] = ' '
                slow += 1

            # 3. 把整个单词搬运过去
            while fast < n and s[fast] != ' ':
                s[slow] = s[fast]
                slow += 1
                fast += 1

        # 4. 继续找下一个单词
        fast += 1

    # 最后 slow 的位置就是新字符串的长度
    return s[:slow]
```

## 1.5 翻转列表

反转链表

题意：反转一个单链表。

示例: 输入: 1->2->3->4->5->NULL 输出: 5->4->3->2->1->NULL

### 1.5.1 思路

利用初始值的值和地址。链表本身的理解会涉及到底层架构：内存的构造是 **地址 + value**，根据地址取值就是指令集合。

两种套路：

- **版本一：双指针法** —— 用 `pre` 和 `cur` 两个指针，遍历一遍把每个节点的 `next` 反转
- **版本二：递归法** —— 不使用循环，函数自身一直递归

### 1.5.2 代码

**版本一：双指针法**

```python
# Definition for singly-linked list.
# class ListNode:
#     def __init__(self, val=0, next=None):
#         self.val = val
#         self.next = next
class Solution:
    def reverseList(self, head: ListNode) -> ListNode:
        cur = head
        pre = None
        while cur:
            temp = cur.next  # 保存一下 cur 的下一个节点，因为接下来要改变 cur->next
            cur.next = pre   # 反转
            # 更新 pre、cur 指针
            pre = cur
            cur = temp
        return pre
```

**版本二：递归法**

```python
# Definition for singly-linked list.
# class ListNode:
#     def __init__(self, val=0, next=None):
#         self.val = val
#         self.next = next
class Solution:
    def reverseList(self, head: ListNode) -> ListNode:
        return self.reverse(head, None)

    def reverse(self, cur: ListNode, pre: ListNode) -> ListNode:
        if cur is None:
            return pre

        temp = cur.next
        cur.next = pre
        return self.reverse(temp, cur)
```

## 1.6 两两交换链表中的节点

给定一个链表，两两交换其中相邻的节点，并返回交换后的链表。

你不能只是单纯的改变节点内部的值，而是需要实际的进行节点交换。

![两两交换示意]({{ '/assets/img/notes/Algorithme/image.png' | relative_url }})

### 1.6.1 思路

采用三个角色：**虚拟头节点**、**curr 操作杆**、**缓存节点**。

- `curr` 是手中唯一的"操作杆"，始终位于要交换的两个节点【之前】
- 其他临时变量（如 `node1`、`node2`）只是为了防止数据在 `next` 重排过程中被擦除

针对链表的两个通用技巧：
1. 用虚拟头节点（dummy）来处理"真正的 head 节点也可能被交换"这种边界情况
2. 在改写 `next` 指针之前，用 `temp` 之类的局部变量先保存目标节点

### 1.6.2 代码

```python
class ListNode:
    def __init__(self, val=0, next=None):
        self.val = val
        self.next = next


class Solution:
    def swapPairs(self, head: ListNode) -> ListNode:
        # 1. 铺路：创建一个虚拟头节点
        # 它的作用是作为"前哨"，帮我们处理真正的 head 节点
        dummy = ListNode(0)
        dummy.next = head

        # curr 是我们的"当前位置"，它始终位于我们要交换的两个节点【之前】
        curr = dummy

        # 2. 检查：只要后面还有两个节点，就可以交换
        while curr.next and curr.next.next:
            # 先给要操作的节点起个名字，防止待会儿"弄丢"
            node1 = curr.next        # 第一个节点
            node2 = curr.next.next   # 第二个节点

            # --- 开始交换（重新连线）---

            # 第一步：让前哨指向第二个节点
            # [curr] -> [node2]
            curr.next = node2

            # 第二步：让第一个节点跨过第二个，指向后面的"剩余部队"
            # [node1] -> [node2.next]
            node1.next = node2.next

            # 第三步：让第二个节点回过头，拉住第一个节点
            # [node2] -> [node1]
            node2.next = node1

            # --- 交换完成 ---

            # 3. 推进：把 curr 移动到交换后的 node1 位置
            # 此时链表变成了 [curr] -> [node2] -> [node1] -> [剩余部队]
            # 为了处理下一对，我们要把 curr 挪到 node1
            curr = node1

        # 返回 dummy.next，也就是新的链表头
        return dummy.next
```

## 1.7 删除链表的倒数第 N 个节点

LeetCode 19. 给你一个链表，删除链表的倒数第 n 个结点，并且返回链表的头结点。

**进阶**：你能尝试使用一趟扫描实现吗？

**示例**：

```text
输入：head = [1,2,3,4,5], n = 2 → 输出：[1,2,3,5]
输入：head = [1],         n = 1 → 输出：[]
输入：head = [1,2],       n = 1 → 输出：[1]
```

### 1.7.1 思路

采用 **虚拟节点 + 快慢双指针** 实现一趟扫描：

1. 创建虚拟头节点 `dummy` 指向 `head`，让 `fast` 和 `slow` 都从 dummy 出发
2. `fast` 先走 n+1 步，使 `fast` 与 `slow` 之间相距 n+1 个节点
3. 同时移动 `fast` 与 `slow`，直到 `fast` 走到 `None`
4. 此时 `slow.next` 正是倒数第 n 个节点，执行 `slow.next = slow.next.next` 删除
