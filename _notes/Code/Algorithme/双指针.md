## 1.1 移除元素

给你一个数组 nums 和一个值 val，你需要 原地 移除所有数值等于 val 的元素，并返回移除后数组的新长度。

不要使用额外的数组空间，你必须仅使用 O(1) 额外空间并原地修改输入数组。

元素的顺序可以改变。你不需要考虑数组中超出新长度后面的元素。

示例 1: 给定 nums = [3,2,2,3], val = 3, 函数应该返回新的长度 2, 并且 nums 中的前两个元素均为 2。 你不需要考虑数组中超出新长度后面的元素。

示例 2: 给定 nums = [0,1,2,2,3,0,4,2], val = 2, 函数应该返回新的长度 5, 并且 nums 中的前五个元素为 0, 1, 3, 0, 4。

你不需要考虑数组中超出新长度后面的元素。

### 1.1.1 思路

由于这里的标题直接是双指针，因为就直接开始使用这个指针进行代码实践

快指针：寻找新数组的元素 ，新数组就是不含有目标元素的数组, 一个用于遍历

慢指针：指向更新 新数组下标的位置，一个用于新数组的下标

```python

class Solution():
    def remove(self, nums: list[int], val: int) -> int:

        fast = 0  # 按照你的习惯命名，但它实际是负责写入的指针
        
        for i in range(len(nums)):
            if nums[i] != val:

                nums[fast] = nums[i]

                fast += 1  # 成功写入一个元素后，指针往前推一格
                
        return fast  # fast 最后的值，恰好就是有效元素的个数

```

不同的写法：

```python

class Solution:
    def removeElement(self, nums: List[int], val: int) -> int:
        # 快慢指针
        fast = 0  # 快指针
        slow = 0  # 慢指针
        size = len(nums)
        while fast < size:  # 不加等于是因为，a = size 时，nums[a] 会越界
            # slow 用来收集不等于 val 的值，如果 fast 对应值不等于 val，则把它与 slow 替换
            if nums[fast] != val:
                nums[slow] = nums[fast]
                slow += 1
            fast += 1
        return slow

```

```python
# 相向双指针法
# 时间复杂度 O(n)
# 空间复杂度 O(1)
class Solution:
    def removeElement(self, nums: List[int], val: int) -> int:
        n = len(nums)
        left, right  = 0, n - 1
        while left <= right:
            while left <= right and nums[left] != val:
                left += 1
            while left <= right and nums[right] == val:
                right -= 1
            if left < right:
                nums[left] = nums[right]
                left += 1
                right -= 1
        return left

```

下面是这个：

首尾呼应，用右边有用的数字，去填补左边需要删除的空位。

1. 左右包抄（初始化）
动作： 定义两个指针，left 站在数组的最前面（索引 0），right 站在数组的最后面（索引 n-1）。

目的： 两人准备相向而行，从两头向中间巡视整个数组。

2. 左指针找“坑”（第一个 while 循环）
动作： 左指针 left 一路向右走。只要它脚下的数字不是目标值 val，说明这个数字是安全的，不用管它。

触发： 一旦 left 踩到了 val，它就立刻停下。

意义： 此时，左边出现了一个需要被填补的**“坑”**（废弃位置）。

3. 右指针找“替补”（第二个 while 循环）
动作： 右指针 right 一路向左（前）退。只要它脚下的数字是目标值 val，说明这本来就是要扔掉的数字，不用管它。

触发： 一旦 right 踩到了一个不是 val 的正常数字，它也立刻停下。

意义： 此时，右边找到了一个可以用来填坑的**“替补”**（有效数据）。

4. 填坑换位（if left < right: 判断与覆盖）
当左边有“坑”，右边有“替补”时，执行终极连招：

第一步（填坑）： 执行 nums[left] = nums[right]，直接用右边的“替补”把左边的“坑”覆盖掉。

第二步（指针推进）： 填完坑后，双方都完成了一次任务，需要各自走向下一个目标：

left += 1（向右走）： 左指针的坑填满了，索引变大，在尺子上向右走，去检查后面下一个未知的元素。

right -= 1（向左走）： 右指针的替补用过了（已经复制到了前面），失去利用价值。索引变小，在尺子上向左走（往前面走），去寻找前一个还没被动过的“新替补”。

## 1.2 反转字符串II

给定一个字符串 s 和一个整数 k，从字符串开头算起, 每计数至 2k 个字符，就反转这 2k 个字符中的前 k 个字符。

如果剩余字符少于 k 个，则将剩余字符全部反转。

如果剩余字符小于 2k 但大于或等于 k 个，则反转前 k 个字符，其余字符保持原样。

示例:

输入: s = "abcdefg", k = 2
输出: "bacdfeg"

## 1.3 替换数字

给定一个字符串 s，它包含小写字母和数字字符，请编写一个函数，将字符串中的字母字符保持不变，而将每个数字字符替换为number。

例如，对于输入字符串 "a1b2c3"，函数应该将其转换为 "anumberbnumbercnumber"。

对于输入字符串 "a5b"，函数应该将其转换为 "anumberb"

输入：一个字符串 s,s 仅包含小写字母和数字字符。

输出：打印一个新的字符串，其中每个数字字符都被替换为了number

样例输入：a1b2c3

样例输出：anumberbnumbercnumber

数据范围：1 <= s.length < 10000

### 1.3.1 思路

按C 或者C++ 来看，首先要考虑使用多少空间,核心思想是从后往前去挪移.

如何是从前方进行的话，会覆盖后面的数据。

两个指针，一个指向数据的末尾，一个指向扩容数据的末尾。

先左边开始扫描，遇到字母就先放后面，始终保存5个空间的位置

替换的时候，本身数字的位置占了一位，所以切片的时候应该

是  s[1:7] 左闭右开

```python

class Solution:
    def replaceNumber(self, s: list) -> list:
        # 1. 统计数字出现的次数
        count = 0
        for char in s:
            if '0' <= char <= '9':
                count += 1
        
        # 记录原始长度
        old_len = len(s)
        # 假设将 1 个数字替换为 "number" (6个字符)，则每个数字需要额外 5 个空间
        extra_space = count * 5
        s.extend([''] * extra_space)
        
        # 2. 定义双指针
        # i 指向原始列表末尾，j 指向扩容后的列表末尾
        i = old_len - 1
        j = len(s) - 1
        
        # 3. 从后往前替换
        while i >= 0:
            if '0' <= s[i] <= '9':
                # 如果是数字，倒序填充 "number"
                for char in reversed("number"):
                    s[j] = char
                    j -= 1
            else:
                # 如果不是数字，直接复制
                s[j] = s[i]
                j -= 1
            i -= 1
            
        return s



```

解释: 如果两个单词间有多余的空格，将反转后单词间的空格减少到只含一个。

### 1.4.1 思路

```python

# 由于输出后的字符串不能有多余的空格，因此，在反转之前，先把这个空格删除.

# 并且需要删除这个两个单词间多余的空格

具体的操作就是先找到非零的单词然后填补上，后面的单词 和单词之间就需要填补一个空格。

def remove_extra_spaces(s: list) -> list:
    slow, fast = 0, 0
    n = len(s)
    
    while fast < n:
        # 1. 寻找单词：跳过所有空格
        if s[fast] != ' ':
            # 2. 如果不是第一个单词，在单词前补一个空格
            if slow != 0:
                s[slow] = ' '
                slow += 1
            
            # 3. 把整个单词搬运过去
            while fast < n and s[fast] != ' ':
                s[slow] = s[fast]
                slow += 1
                fast += 1
        
        # 4. 继续找下一个单词
        fast += 1
        
    # 最后 slow 的位置就是新字符串的长度
    return s[:slow]

```

### 1.5.1 思路

利用初始值的值和地址

链表本身的理解会涉及到这个底层架构

内存是的构造 是一个地址 ＋ value

根据地址取值， 指令集合

```python

（版本一）双指针法
# Definition for singly-linked list.
# class ListNode:
#     def __init__(self, val=0, next=None):
#         self.val = val
#         self.next = next
class Solution:
    def reverseList(self, head: ListNode) -> ListNode:
        cur = head   
        pre = None
        while cur:
            temp = cur.next # 保存一下 cur的下一个节点，因为接下来要改变cur->next
            cur.next = pre #反转
            #更新pre、cur指针
            pre = cur
            cur = temp
        return pre

```

第二个套路是使用递归的思想, 不使用很多循环，使用函数一直进行递归。

```python

（版本二）递归法
# Definition for singly-linked list.
# class ListNode:
#     def __init__(self, val=0, next=None):
#         self.val = val
#         self.next = next
class Solution:
    def reverseList(self, head: ListNode) -> ListNode:

        return self.reverse(head, None)


    def reverse(self, cur: ListNode, pre: ListNode) -> ListNode:

        if cur == None:
            return pre

        temp = cur.next

        cur.next = pre

        return self.reverse(temp, cur)

```

## 1.6 两两交换链表中的节点

给定一个链表，两两交换其中相邻的节点，并返回交换后的链表。

你不能只是单纯的改变节点内部的值，而是需要实际的进行节点交换。

![alt text](../../../assets/img/notes/Algorithme/image.png)

思路采用的是三个节点，虚拟节点和缓存节点

curr 是你手中唯一的“操作杆”，而其他的临时变量（如 node1, node2）本质上就是,  因为数据可能会被擦除。

针对链表的使用一个是采用虚拟节点，另外就是找temp 去保存。

```python
class ListNode:
    def __init__(self, val=0, next=None):
        self.val = val
        self.next = next

class Solution:
    def swapPairs(self, head: ListNode) -> ListNode:
        # 1. 铺路：创建一个虚拟头节点。
        # 它的作用是作为“前哨”，帮我们处理真正的 head 节点
        dummy = ListNode(0)
        dummy.next = head
        
        # curr 是我们的“当前位置”，它始终位于我们要交换的两个节点【之前】
        curr = dummy
        
        # 2. 检查：只要后面还有两个节点，就可以交换
        while curr.next and curr.next.next:
            # 先给要操作的节点起个名字，防止待会儿“弄丢”
            node1 = curr.next       # 第一个节点

            node2 = curr.next.next  # 第二个节点
            
            # --- 开始交换（重新连线）---
            
            # 第一步：让前哨指向第二个节点
            # [curr] -> [node2]
            curr.next = node2
            
            # 第二步：让第一个节点跨过第二个，指向后面的“剩余部队”
            # [node1] -> [node2.next]
            node1.next = node2.next
            
            # 第三步：让第二个节点回过头，拉住第一个节点
            # [node2] -> [node1]
            node2.next = node1
            
            # --- 交换完成 ---
            
            # 3. 推进：把 curr 移动到交换后的 node1 位置
            # 此时链表变成了 [curr] -> [node2] -> [node1] -> [剩余部队]
            # 为了处理下一对，我们要把 curr 挪到 node1
            curr = node1
            
        # 返回 dummy.next，也就是新的链表头
        return dummy.next


    



```
